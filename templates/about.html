About TransLink Performance Explorer

What is TransLink Performance Explorer?

TransLink Performance Explorer is a web application designed to help users analyze and visualize transit performance data. It fills in the gap found in TransLink's official TSPR reports and summaries by allowing users to easily explore and compare performance metrics between each other as well as create custom visualizations. The app is built using Python and Flask, and it utilizes the TSPR CSV files provided by TransLink as its data source.
