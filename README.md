# TransLink-Performance-Explorer

Live Project: Coming soon

## What is TransLink-Performance-Explorer?
 A performance data analyzer that generates comparisons and infographics of bus lines/SkyTrain stations you choose to compare.

## Notes

● Only "My 2 Buses" are implemented as of now. Rest are coming soon.

## How to use (as of now)

1. Download files to your computer

2. On terminal, navigate to the file "TransLink-Performance-Explorer"

3. Type command: python app.py

4. On browser, open: http://127.0.0.1:5000/

5. Use as a webapp!
